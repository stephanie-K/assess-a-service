# assess-a-service

This is a personal project. I'm trying to offer a tool with various options to assess a service using existing frameworks and tools publicly available:

- GOV.UK Service standard
- Scottish Government Service standard
- Good service scale (by Lou Downe)
- Ethics assessement  (by spotify.Design)


**This is under construction**

[You can visit the website](https://assess-a-service.herokuapp.com/)
